# 机器学习的学习之路

>参考内容：[神经网络和深度学习](https://mooc.study.163.com/course/2001281002#/info)、[相关内容（深度学习工程师）](https://mooc.study.163.com/smartSpec/detail/1001319001.htm)

- 阶段3：库TensorFlow以及Keras等，了解内容，能够自己写

>参考内容：[TensorFlow官方文档](http://wiki.jikexueyuan.com/project/tensorflow-zh/)
